# 🌿 Luo World

A photorealistic 3D open-world survival crafting game.

**Engine:** Godot 3.2.3
**Style:** Sons of the Forest / photorealistic jungle
**Perspective:** 3rd person
**Platform:** Android (APK) + Desktop

## Download APK
See [Releases](https://github.com/luokai25/Luo-world-/releases) for the latest APK.

## Controls
| Control | Action |
|---------|--------|
| Left joystick | Move |
| Right joystick | Camera |
| ⚔ | Attack / Chop / Hunt |
| E | Interact / Pick up / Fill cup |
| USE | Eat / Drink |
| 🎒 | Inventory |
| ↑ | Jump |

## World
- 512×512 jungle · 5 biomes · 600 trees · 300 rocks
- River + 8 ponds · 195 animals
- Campfire system · weather · day/night
- 15+ crafting recipes
